A concept-driven portfolio website designed for a **.NET Developer** and **Digital Marketing Executive**. 

This project moves away from standard corporate templates, instead using a **"Digital Court"** theme that reflects my passion for Basketball, Art, and Technology. It treats code like a sport: structured, fast, and strategic.

## üåü Key Features

* **Dual-Identity Hero:** Immediately establishes expertise in both Backend Development and Marketing Strategy.
* **Thematic UI:** A "Dark Mode" interface with basketball court geometry, energetic orange accents, and artistic brush-stroke reveals.
* **Zero Dependencies:** Built entirely with vanilla HTML, CSS, and JavaScript. No heavy frameworks or build steps required.
* **Performance Animations:**
    * **Scroll Reveals:** Uses the `IntersectionObserver` API for smooth element fade-ins.
    * **Fusion Zone:** An accordion-style interest section that expands on hover.
    * **Physics Hovers:** Buttons and icons react with "dribble" and rotation effects.
* **Fully Responsive:** optimized for mobile, tablet, and desktop screens.

## üöÄ Quick Start

Since this is a static site, no installation is required!

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/satish051/The-Digital-Playbook.git](https://github.com/satish051/The-Digital-Playbook.git)
    ```
2.  **Open the site:**
    Simply double-click `index.html` to open it in your browser.

## üõ†Ô∏è Customization

To make this your own, look for the following sections in `index.html`:

* **Images:** Replace the URL placeholders in the `.fusion-bg` divs (Line ~330) with your own images hosted on Imgur or in an `/assets` folder.
* **Contact Info:** Update the `mailto:` and `tel:` links in the footer section.
* **Colors:** You can change the entire theme by modifying the CSS Variables at the top of the `<style>` block:
    ```css
    :root {
        --accent-bball: #ff5e00; /* Change for primary accent */
        --accent-art: #00f2ff;   /* Change for secondary accent */
    }
    ```

## üì° Deployment

This site is ready for **GitHub Pages**.

1.  Go to your repository **Settings**.
2.  Click **Pages** on the left sidebar.
3.  Under **Source**, select `Deploy from a branch`.
4.  Select `main` (or `master`) and `/root`, then click **Save**.

Your site will be live at `https://satish051.github.io/The-Digital-Playbook/`.
